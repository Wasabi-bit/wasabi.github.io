# 家庭账本设计说明

## 设计目标

这个 App 的视觉目标是：简单、好记、像一个可以长期使用的个人工具。

参考方向不是复制 Airbnb、YouTube 或 Bolt 的具体图形，而是学习它们的共同点：

- 图标轮廓清楚
- 元素少
- 主色明确
- 远看也能认出来
- 不依赖复杂装饰

## 当前默认图标

默认使用方案 B：语音记账。

原因：

- 一眼能看出和语音输入有关。
- 红色更醒目，放在手机桌面上更容易记住。
- 气泡和麦克风能突出“说一句就记账”的核心功能。

## App UI 方向

- 主色：账本绿 `#126b5a`
- 记忆点：目标金 `#ffcf57`
- 卡片圆角：保持 8px，像工具而不是玩具
- 页面结构：首页、记账、统计、成员四个标签
- 交互反馈：底部标签有轻微回弹动画
